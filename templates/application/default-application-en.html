Configuration options (GCWeb Jekyll specific)

  • "applicationName" (mandatory, Object): contains the application name for all languages.
    • "en" (mandatory): the English application name.
    • "fr" (mandatory): the French application name.
  • "applicationURL" (mandatory, Object): contains the application URL for all languages.
    • "en" (mandatory): the English application URL.
    • "fr" (mandatory): the French application URL.

Code

{
	...
	"applicationName": {
		"en": "My application",
		"fr": "Mon application"
	}
	"applicationURL": {
		"en": "/",
		"fr": "/"
	}
	...
}
