Options de configuration (spécifique à GCWeb Jekyll)

  • "applicationName" (obligatoire, Object): contient le nom de l'application pour toutes les langues.
    • "en" (obligatoire): le nom de l'application en anglais.
    • "fr" (obligatoire): le nom de l'application en français.
  • "applicationURL" (obligatoire, Object): contient l'URL de l'application pour toutes les langues.
    • "en" (obligatoire): l'URL de l'application en anglais.
    • "fr" (obligatoire): l'URL de l'application en français.

Code

{
	...
	"applicationName": {
		"en": "My application",
		"fr": "Mon application"
	}
	"applicationURL": {
		"en": "/",
		"fr": "/"
	}
	...
}
